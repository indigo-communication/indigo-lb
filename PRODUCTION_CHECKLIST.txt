## ğŸ› TROUBLESHOOTING GUIDE

### Contact Form Not Sending Emails:

**Issue:** Form submits but no email received

**Possible Causes:**
1. Email in spam folder
2. PHP mail() function disabled
3. Server email configuration issue
4. Indigo platform form backend issue

**Solutions:**
1. Check spam/junk folder
2. Wait 5-10 minutes for email delivery
3. Ask Namecheap support to verify PHP mail() is enabled
4. Contact form uses Indigo platform backend (www.imdomainrouter.com/form_submit) - should work automatically

---

### HTTPS Not Working:

**Issue:** Site shows "Not Secure" or doesn't load with HTTPS

**Possible Causes:**
1. SSL certificate not activated yet
2. Waiting for DNS propagation
3. .htaccess not loaded

**Solutions:**
1. Wait 24 hours for SSL activation
2. Clear browser cache (Ctrl+Shift+R)
3. Contact Namecheap support to activate SSL
4. Verify .htaccess file uploaded correctly

---

### Clean URLs Not Working (404 Errors):

**Issue:** /about-indigo shows 404 error

**Possible Causes:**
1. .htaccess not loaded
2. mod_rewrite not enabled
3. File permissions incorrect

**Solutions:**
1. Verify .htaccess exists in public_html
2. Contact Namecheap to enable mod_rewrite
3. Temporarily use full URLs: /about-indigo.html
4. Check .htaccess permissions (should be 644)

---

### Images Not Loading:

**Issue:** Broken image icons or missing images

**Possible Causes:**
1. Incorrect file paths (case sensitivity)
2. File permissions too restrictive
3. images/ folder not uploaded

**Solutions:**
1. Verify images/ folder exists in public_html
2. Set folder permissions to 755
3. Set image file permissions to 644
4. Check for case-sensitive filename issues (Linux hosting)
